# RDK location-based tutorial

Welcome to the RDK location-based AR tutorial! This tutorial will get you started on the basics of setting up a location-based AR app using the RDK framework. You will start with a "Hello World" app and progress on to a full server-backed app displaying POIs and roads, sourced from OpenStreetMap.

The following tutorials are currently complete but will be added to later.

- [Part 1](part1.md) : Hello World!
- [Part 2](part2.md) : Fetching data from a server
- [Part 3](part3.md) : Storing data in a database
- [Part 4](part4.md) : Rendering roads and paths
